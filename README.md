# ­ЪЏа№ИЈ dev-tools-hub

`dev-tools-hub` рЃљрЃарЃўрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃДрЃћрЃюрЃћрЃЉрЃћрЃџрЃў CLI рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃЦрЃЏрЃюрЃўрЃА Vite + React + Tailwind CSS рЃърЃарЃЮрЃћрЃЦрЃбрЃА TypeScript-рЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃўрЃЌ, рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃћрЃЉрЃўрЃЌ рЃЊрЃљ рЃњрЃљрЃФрЃџрЃћрЃЋрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљрЃА рЃЊрЃљрЃљрЃЏрЃљрЃбрЃЮ рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃў рЃћрЃарЃЌрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃЌ.

---

## Рюе рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў

- ­ЪћД рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃЦрЃЏрЃюрЃўрЃА Vite + React + TypeScript рЃърЃарЃЮрЃћрЃЦрЃбрЃА
- ­Ъје рЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃА Tailwind CSS-рЃА рЃЊрЃљ рЃљрЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљрЃА
- ­ЪЌѓ№ИЈ рЃЦрЃЏрЃюрЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџ рЃцрЃЮрЃџрЃЊрЃћрЃа рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃА (`components`, `pages`, `hooks`, рЃЊрЃљ рЃљ.рЃе.)
- ­Ъїѕ рЃљрЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃА Tailwind рЃцрЃћрЃарЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃбрЃЮрЃарЃА
- ­ЪЊд рЃњрЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃўрЃарЃЕрЃўрЃЮ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃў (рЃЏрЃљрЃњ: `axios`, `react-router-dom`, `zustand`)
- ­ЪД╣ рЃерЃџрЃўрЃА рЃќрЃћрЃЊрЃЏрЃћрЃб рЃцрЃљрЃўрЃџрЃћрЃЉрЃА (`App.css`, `index.css`)
- ­ЪЉІ рЃљрЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃћрЃЉрЃА App.tsx-рЃА рЃерЃћрЃюрЃў рЃърЃћрЃарЃАрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃЏрЃўрЃАрЃљрЃџрЃЏрЃћрЃЉрЃўрЃЌ

---
